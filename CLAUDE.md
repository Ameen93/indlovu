# CLAUDE.md - indlovu

## Project
Privacy-first vector database with built-in POPIA/GDPR compliance. Rust-based.

## Stack
Rust, Cargo workspace

## Commands
- `cargo build` — build all crates
- `cargo test` — run tests

## Structure
- `crates/` — workspace crates (core, compliance, server, python bindings)
- `site/` — landing page
- `examples/` — usage examples
